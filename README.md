# file-encryption
Disclaimer: This is a hobby project and has not been audited for issues.

Encryption: AES-256-CBC
Authentication: HMAC-SHA256 Encrypt-then-MAC
Key derivation: Argon2id with parameters memory=2GiB, iterations=2, threads=4
